# 🗣️ Command Reference

Control your drone using natural language. The system recognizes the following intents and phrases.

## 🛫 Basic Actions

| Action | Voice Commands |
| :--- | :--- |
| **Takeoff** | "Take off 10", "Takeoff 5 meters" |
| **Land** | "Land", "Land now" |
| **Return** | "Return to Launch", "RTL", "Return Home", "Go Home" |
| **Arm** | "Arm the drone" |
| **Disarm** | "Disarm" |

## ↔️ Movement

| Action | Voice Commands |
| :--- | :--- |
| **Forward** | "Move forward 5 meters", "Go forward 10" |
| **Backward** | "Move backward 5", "Go back" |
| **Left** | "Move left 5 meters", "Go left" |
| **Right** | "Move right 5 meters", "Go right" |
| **Up** | "Move up 2 meters", "Ascend 5" |
| **Down** | "Move down 2 meters", "Descend" |

## 🔄 Orientation

| Action | Voice Commands |
| :--- | :--- |
| **Rotate Right** | "Rotate right 90 degrees", "Turn right" |
| **Rotate Left** | "Rotate left 45 degrees", "Turn left" |

## 🛑 Safety

| Action | Voice Commands |
| :--- | :--- |
| **Hold/Stop** | "Hold position", "Stop", "Hover" |

## 💡 Tips

-   **Be Specific**: Adding numbers helps precision (e.g., *"Take off **10 meters**"*).
-   **Defaults**: If you don't say a number, the system uses safe defaults (e.g., Takeoff = 10m, Move = 5m).
